# 💰 My Wallet - ระบบบันทึกรายรับ-รายจ่าย

เว็บไซต์สำหรับบันทึกรายรับและรายจ่ายส่วนบุคคล ออกแบบด้วยสไตล์มินิมอล ใช้งานง่าย รองรับการแสดงผลทั้งบนคอมพิวเตอร์และมือถือ (Responsive Design)

## 🌐 ลิงก์เข้าใช้งานเว็บไซต์
👉 [คลิกที่นี่เพื่อเปิดใช้งานเว็บไซต์](https://pure3c.github.io/monyflow/)

## ✨ ฟีเจอร์เด่น (Features)
- **Dashboard สรุปยอด:** แสดงยอดรายรับรวม, รายจ่ายรวม และยอดเงินคงเหลือสุทธิแบบ Real-time
- **ระบบบันทึกรายการ:** แยกฝั่งรายรับ (+) และรายจ่าย (-) ชัดเจนด้วยสีเขียวและสีแดง
- **ประวัติรายการ (History):** แสดงรายการล่าสุดเรียงลงมา พร้อมปุ่มลบรายการที่ไม่ต้องการ
- **ระบบจำข้อมูล:** ใช้ `LocalStorage` ทำให้ข้อมูลไม่หายไปแม้ว่าจะกดปิดหรือรีเฟรชหน้าเว็บ

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)
- HTML5
- CSS3 (Custom Variables & Flexbox/Grid)
- JavaScript (Vanilla JS)

## 💾 ระบบการจัดเก็บข้อมูล
ข้อมูลทั้งหมดจะถูกบันทึกไว้ในเบราว์เซอร์ของเครื่องผู้ใช้งานเองผ่าน **Web Storage (LocalStorage)** ไม่มีการส่งข้อมูลไปยังเซิร์ฟเวอร์ภายนอก ปลอดภัย 100%
